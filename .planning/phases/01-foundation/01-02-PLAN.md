**data/playbook.md** (per D-07, D-08):

Create a markdown document with scoring guidance for all 8 rubric items. Structure it with a section per item (## Item 1a, ## Item 1b, etc.).

For each item, provide Score 0, Score 1, and Score 2 descriptions.

**Extraction items (1a, 2a, 3a, 4a) -- PRECISE guidance (D-07):**
The 0/1/2 boundary must be unambiguous. Example pattern:
- "Score 0 if [specific thing] is not mentioned."
- "Score 1 if [partial identification described specifically]."
- "Score 2 if [exact identification criterion]."

Specific guidance per extraction item:
- **1a:** Score 0 if confidentiality period not mentioned. Score 1 if a duration is mentioned but not the specific 7-year term. Score 2 if the 7-year term is explicitly identified.
- **2a:** Score 0 if the definition of Confidential Information is not discussed. Score 1 if the definition is mentioned but its scope is not described. Score 2 if the review quotes or specifically describes the breadth of categories covered (business, operations, customers, technology, strategy).
- **3a:** Score 0 if non-solicitation is not mentioned. Score 1 if non-solicitation is mentioned but the 24-month duration or its location in the agreement is not specified. Score 2 if the 24-month non-solicitation obligation is identified and its location in the remedies section is noted.
- **4a:** Score 0 if the independently-developed-information exception is not mentioned. Score 1 if the exception is mentioned but the qualifying requirements are not described. Score 2 if the exception is identified and the specific onerous requirements (contemporaneous documentation, audit, prior notice, etc.) are described.

**Judgment items (1b, 2b, 3b, 4b) -- DELIBERATELY VAGUE guidance (D-07, D-08):**
The 0/1/2 boundary must be harder for the judge to apply. Use language like "substantive assessment", "meaningful analysis", "practical implications". Example pattern:
- "Score 0 if [not discussed at all]."
- "Score 1 if [mentioned but not assessed] -- this boundary is intentionally soft."
- "Score 2 if [substantive assessment provided]."

Specific guidance per judgment item:
- **1b:** Score 0 if the duration is not discussed in terms of market norms. Score 1 if market comparison is mentioned but without substantive analysis. Score 2 if the review provides a substantive assessment of how the 7-year term compares to market norms and the practical implications.
- **2b:** Score 0 if the breadth of the definition is not assessed as a risk. Score 1 if overbreadth is noted but not analysed in terms of negotiation impact. Score 2 if the review provides a meaningful analysis of the negotiation risk posed by the definition's scope.
- **3b:** Score 0 if the placement and scope implications are not discussed. Score 1 if the review notes something about the clause but does not connect placement to scope concerns. Score 2 if the review provides a substantive assessment of why a non-solicitation in the remedies section is unusual and how the obligation extends beyond typical NDA scope.
- **4b:** Score 0 if the practical effect of the requirements is not discussed. Score 1 if the review mentions the requirements but does not assess their cumulative effect. Score 2 if the review provides a substantive assessment of how the additional requirements render the exception functionally unusable.

The playbook should have a brief introduction explaining its purpose, then the 8 item sections.
